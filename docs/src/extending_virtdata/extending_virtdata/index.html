Extending VirtData

The VirtData library is not meant to be a closed system. Much of the runtime wiring was built to allow easy extension with data mapping functions of your own design. This is true at the module level as well as the individual function level.

The following sections should tell you everything you need to customize the avaialble data mapping functions for your own projects.
